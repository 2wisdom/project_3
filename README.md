## 프로젝트 소개

---

이미지로 식물 예측이 가능한 온라인 커뮤니티 웹 사이트 제작

<br>

## 주요 기능

---

1. 식물 이미지 검색
2. 커뮤니티 게시글 및 댓글 작성
3. 식물 관련 상품 거래
